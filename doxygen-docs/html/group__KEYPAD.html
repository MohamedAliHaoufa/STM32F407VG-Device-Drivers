Scan and print the pressed key on the keypad.

This function scans a specific row of the keypad for keypress and prints the pressed key on an LCD.

Parameters
[in]row_numberThe row number to scan (0 to 3 for a 4x4 keypad).
[in]row_pinThe GPIO pin number connected to the row.
[in]column_pinThe GPIO port connected to the column.
[in]pGPIOxPointer to the GPIO peripheral associated with the keypad.
